# Codex 指南

Codex 是面向代码任务的 Agent/CLI，可用于读写项目、执行命令并协助完成开发工作流。这页目前还是占位页，后续会按安装、配置、工作流和常见问题逐步补全。

Codex 安装推荐通过 Microsoft Store 安装。

推荐使用 GPT 模型时使用 Codex。

## 全局指令

AGENTS.md 是 Codex 读取的项目/用户级智能体指令文件，作用是告诉编码智能体“项目怎么构建、测试、改代码、遵守哪些规范”；通常放在项目根目录，必要时也可放在子目录做局部规则，用户全局规则则分别放在对应工具支持的用户配置目录中。

本人使用指令文件参考：

[AGENTS.md](./AGENTS.md)

可以下载后放到 `%USERPROFILE%/.codex` 下或者项目根目录的 `.codex` 文件夹下。
