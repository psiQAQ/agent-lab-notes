# 仓库级 Agent 指令

编辑本仓库文档前，先读取并遵守根目录 [CLAUDE.md](./CLAUDE.md) 中的写作偏好。

## 正文与附录

- 正文应保持安装、配置和使用流程连续。读者按顺序操作时，不应被长篇背景或实验说明打断。
- 背景、设计依据、实验方法、完整数据、可靠性说明和适用范围较长时，移到文末附录。
- 正文只保留理解或操作所需的简短摘要，并用清晰的相对链接指向附录。
- 不要为了缩短正文删除验证证据；调整信息层级和位置。
- 安装前提、必要命令、安全警告以及直接影响操作结果的限制仍放在正文。
- 新增内容时先判断读者是否必须在当前步骤知道：必须知道就留在当前位置，否则优先放入附录。
